#
游戏玩法介绍：
<img width="1909" height="1125" alt="image" src="https://github.com/user-attachments/assets/6bf4d129-fb25-4479-986e-65db67b39849" />
开始界面：
按下“start”进入游戏，按下"quit"结束程序
<img width="1907" height="1125" alt="image" src="https://github.com/user-attachments/assets/7bacba95-e79c-4e2e-a5b9-33503744785c" />
游戏内操作：
按下WASD可移动“龙”，按J释放普攻，按K释放技能（需要能量，能量会自动回复）
按下P或ESC可暂停游戏
如果按下对应按键无反应，可按shift并再次尝试
<img width="1915" height="1126" alt="image" src="https://github.com/user-attachments/assets/ad11847d-1f67-4622-b059-17b6fc316e69" />
暂停界面：
按下“Back”会回到游戏，按下"Quit"会回到开始界面
<img width="1915" height="1121" alt="image" src="https://github.com/user-attachments/assets/90ae0445-3a4c-4464-bbe2-acd086d24b95" />
结算界面：
击败游戏内所有敌人或“家”的血量为0时游戏结束，按下"ESC"可回到开始界面
